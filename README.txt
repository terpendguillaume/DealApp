BASE DE DONNE: Sqlite
SERVER: Node.js
APPLICATION: Ionic
__________________________________________________________________________________
Comment lancer l'application ?
> Installer Node.js
> Lancer le server: node server.js (dans le repertoire api)
> Installer Ionic: npm install -g cordova ionic
> Lancer l'application: ionic serve (dans le repertoire de l'application)
OU
> Executer le shell: start.sh
__________________________________________________________________________________
FONCTIONNEMENT:
L'application permet de gérer les coupons de réductions
Celle-ci est utlisée par deux types d'utilisateurs: clients & gérants

Sur la page d'accueil, 2 onglets présentant soient les coupons populaires, 
soient l'ensemble des coupons sont disponibles.

Via un Side-menu, on peut accéder à notre compte utilisateur (pour se connecter).
Se connecter à l'applciation permet d'acceder à deux nouvelles fonctionnalités.
Un onglet pertmettant d'afficher l'ensemble des coupons associé au compte.
Un onglet d'échange permettant aux client d'échanger leurs coupons entre eux.
